# Crop Prediction Web App

## Project Description

The **Crop Prediction Web App** is a responsive web application designed to help farmers and agronomists predict the most suitable crop for their land based on soil properties, nutrient composition, and local weather conditions. By entering key soil metrics such as pH, nitrogen, phosphorus, potassium, zinc, and sulfur (all in mg/kg), along with rainfall, temperature, and humidity, the application ensures that users provide valid and accurate data before performing predictions.

This project is developed as part of my **research mentorship at Wale Lab**, where I focus on applying data-driven approaches and machine learning techniques to agriculture and soil analysis. The mentorship emphasizes practical applications, user-centered design, and clean, reproducible workflows.

The app is built with a clean and intuitive user interface, featuring grouped input sections for soil properties, nutrients, and weather, along with client-side validation to prevent incorrect entries. A "Back to Home" button ensures easy navigation, and the layout is mobile-friendly for on-the-go usage.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend (optional):** Python, Flask  
- **Libraries:**  Flask-WTF, pandas, joblib, scikit-learn  
- **Version Control:** Git, GitHub  

## Key Features
- Input soil pH, color, nutrients (mg/kg), and weather conditions.
- Client-side validation ensures all fields are correctly filled before prediction.
- Responsive design for both desktop and mobile devices.
- Back to Home button for smooth navigation.
- Ready to integrate with a Python-based prediction model.
