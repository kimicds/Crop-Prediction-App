About the Project

Project Purpose

This project is an agricultural decision support system developed to help farmers make informed crop selection decisions based on soil properties, climate conditions, and environmental factors.

Problem It Addresses

Many farmers rely on experience alone when choosing crops, despite variations in soil nutrients, rainfall, temperature, and humidity. This system reduces uncertainty by providing data-backed predictions.

How the System Works

The platform applies machine learning model to analyze soil and weather data, identifying patterns that influence crop suitability and generating predictions aligned with real farming conditions.

Mentorship & Development

This project was developed under the Wale lab mentorship program supervised by Mr. Chibuzor, with a strong focus on applying data science to practical agricultural challenges and real-world impact.

Implementation Approach

The system is built with a lightweight Flask architecture for ease of use and deployment. Model retraining is handled externally to ensure reliability, stability, and controlled performance.

Overall Goal

The goal is to support smarter farming decisions, improve productivity, and help farmers plan with confidence using data-driven insights.
